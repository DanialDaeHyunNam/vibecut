## Decisions & Constraints
- **구독 우선, 토큰 불가침**: 사용자 본인 구독/키로 인증, 앱은 토큰을 절대 다루지 않음 — 모든 프로바이더 결정의 기준
- **Gemini만 API key**: Google이 서드파티 Gemini CLI OAuth 금지·유료 계정 정지 → AI Studio 키로 전환(구독 인증 의도적 거부)
- **원격 정책 킬스위치**: 2026년 프로바이더 약관 3회 변동 → 재배포 없이 프로바이더를 끌 수 있는 정적 매니페스트 도입
- **브랜딩 준수**: 제품 내 "Claude Code" 표기 금지(파트너 가이드) → 라벨 "Claude"
- **제약**: Agent SDK는 MIT 아님(패키징 동봉 약관 B4 확인 필요); Windows 실기기 미검증; Decart Lucy Pro ~$0.15/s 비용 명시 필요; HUD↔에디터 단일 윈도우라 자기 UI 녹화 불가(데모는 OS 캡처 우회)
